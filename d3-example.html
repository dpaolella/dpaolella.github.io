Altair D3

An example interactive plot made using d3

**This plot is currently a work in progress.**

This weighted voronoi treemap[1] shows total CO2 emissions and CO2 emissions per capita by country in 2014.[2] Each polygon represents a different country nested within continent groupings. The size of a given polygon is proportional to the country's total CO2 emissions, and the color represents per capita CO2 emissions. Hover over a polygon to see the country's name and per capita CO2 emissions value.

[1] Code is based off of work by Kcnarf

[2] Data from The World Bank
